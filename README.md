# Desain Diagram

Pada diagram tersebut ada 5 class yaitu Person, Perawat, Pasien, Dokter, dan Hospital. Memiliki 2 inthetance yaitu Person, Perawat, Dokter, dan Pasien, untuk composition yaitu ada di Hospital, Person mewarisi 3 class yaitu Perawat, Dokter, dan Pasien. Untuk composition yaitu Hospital. Jika Hospital dihapus dari sistem, maka seluru Pasien, Dokter, dan Perawat juga ikut menghilang 

# Cara Menambahkan Data

Untuk menambahkan pada data tersebut dengan statis atau di hardcode pada Main, bisa menambahkan Dokter, Perawat, dan Pasien yang sama yaitu, Nama, Nomor Telp, Alamat, id (masing masing class), jika di 
Dokter ada 2 atribut yang berbeda yaitu: Spesialis dan jadwal_praktek.
Pasien ada 2 atribut yang berbeda yaitu: penyakit dan status_rawat
Perawat ada 2 atribut yang berbeda yaitu: shift dan ruangan
